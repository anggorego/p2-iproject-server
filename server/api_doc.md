# GYM-FREAKS API Documentation

## Endpoints :

List of available endpoints:

- `POST /register`
- `POST /login`
- `GET /threads`
- `GET /threads/:id`
- `POST /threads`
- `DELETE /threads/:id`
- `UPDATE /threads/:id`
- `GET /categories`
- `POST /categories`
- `GET /histories`
- `DELETE /histories/:id`
- `POST /pub/register`
- `POST /pub/login`
- `GET /pub/products`
- `GET /pub/products/:productId`

- `GET /pub/favorite`
- `POST /pub/favorite`
